## 一、项目介绍

开发语言：java

运行环境:idea或eclipse 数据库:mysql

技术栈：spring 、spring MVC 、mybatis、jsp

三个角色:管理员、快递员、用户

本系统实现了三大模块：

管理员模块:具备快递人员管理、物流订单管理、取件发件管理以及系统其他配置功能；

用户模块:具备发布代取件、送件信息管理、评价功能和查询订单功能等；

快递模块:具备物流订单管理功能和公告功能
